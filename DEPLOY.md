# 部署 GitHub Pages - 需要认证

## 步骤 1: 认证 GitHub CLI

请在终端执行以下命令：

```bash
gh auth login
```

然后按照提示：
1. 选择 GitHub.com
2. 选择 HTTPS
3. 选择 Login with a web browser
4. 按下 Enter 复制 code
5. 在浏览器粘贴 code 并授权

## 步骤 2: 确认认证

认证完成后，运行：

```bash
gh auth status
```

## 步骤 3: 部署

认证成功后，我将继续部署。
